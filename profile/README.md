### What is Soma?

An AI coding agent with self-growing memory. Every session she learns — patterns become muscles, muscles become skills, skills become instinct. Built on [pi](https://github.com/badlogic/pi-mono).

### Three Layers

| Layer | What | Examples |
|-------|------|---------|
| **⟐ Extensions** | TypeScript hooks into the agent lifecycle | boot sequences · branded headers · auto-flush |
| **◈ Skills** | Markdown knowledge loaded on demand | logo design · social previews · favicon generation |
| **◎ Rituals** | Multi-step workflows as slash commands | `/publish` · `/molt` · `/patrol` |

### Quick Start

```bash
npm i -g meetsoma
soma
```

On first run, Soma creates `.soma/` and writes her own identity based on your workspace. Next session, she picks up where she left off.
